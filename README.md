# YoloFletDemo
物体検出をGUIで行えるPython製アプリ
<br>

## Getting Started

### 1. Pythonの仮想環境を作成
```
python -m venv .venv
```


### 2. 仮想環境を有効化

#### Windows
```
.venv\Scripts\activate
```

#### macOS
```
source .venv/bin/activate
```


### 3. パッケージをインストール
```
pip install -r requirements.txt
```


### 4. fletアプリを立ち上げ
```
flet run
```
